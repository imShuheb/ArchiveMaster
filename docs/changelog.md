## [v1.0.0] - 2026-05-30

### 🎉 Initial Release
This is the first official release of ArchiveMaster!

### Features
- **Smart Deduplication**: Identifies exact file matches using SHA-256 hashing.
- **Granular Media Control**: Toggle between scanning `Photos`, `Videos`, or `Others`.
- **EXIF Date Extraction**: Intelligently reads metadata embedded inside your photos for precise chronological sorting.
- **Modern UI**: A beautiful native desktop interface built with Fluent Design principles.
- **Session History**: Keeps a persistent log of your past organization runs.
- **Automated CI/CD**: Fully automated pipeline producing Windows Installers and Linux Binaries.
