# 📥 Downloads

Download the latest version of ArchiveMaster below. 

All releases are open-source and automatically compiled via GitHub Actions to guarantee safety and transparency.

---

## 🪟 Windows

We offer two versions for Windows: a standard Installer (recommended) and a Portable ZIP.

### 1. Windows Installer (Recommended)
This is the standard Windows Setup executable. It will install ArchiveMaster, create a Start Menu shortcut, and handle everything for you.

- **[Download `ArchiveMaster_Setup.exe`](https://github.com/imShuheb/ArchiveMaster/releases/latest/download/ArchiveMaster_Setup.exe)**

### 2. Windows Portable
A standalone `.zip` file containing the application. No installation is required. Just extract the folder and double-click `ArchiveMaster.exe`.

- **[Download `ArchiveMaster-Windows-Portable.zip`](https://github.com/imShuheb/ArchiveMaster/releases/latest/download/ArchiveMaster-Windows-Portable.zip)**

---

## 🐧 Linux

A standalone `.tar.gz` archive containing the Linux binary.

1. Download the archive below.
2. Extract the contents: `tar -xvf ArchiveMaster-Linux-Portable.tar.gz`
3. Run the binary: `./ArchiveMaster`

- **[Download `ArchiveMaster-Linux-Portable.tar.gz`](https://github.com/imShuheb/ArchiveMaster/releases/latest/download/ArchiveMaster-Linux-Portable.tar.gz)**
